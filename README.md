# 2D-Game-Development-Tower-Of-Melodies

## 2D Game Development module project 

I developed a 2D video game prototype in which the player takes the role of a girl attempting to escape a wizard tower by collecting keys required to unlock locked doors. In addition to searching for keys, the game features "lock" puzzles where the player must click on glowing spots at the correct moment as a slider passes over them.
